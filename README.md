# Programacion_Dinamica
1. Optimización de cortes (cut_rod)
Aquí hacemos un ejercicio de maximización de ganancias:

Definimos el problema: Tenemos una barra de metal de una longitud "n" y una lista de precios para cada tamaño posible de corte.

Buscamos la mejor opción: El algoritmo prueba todas las combinaciones de cortes para decirnos cuál nos deja más dinero.

Resultado en tu ejecución: Para una barra de tamaño 4, el sistema determinó que el "Corte óptimo" es hacer 4 piezas de tamaño 1 (1 1 1 1).

2. Comparación de secuencias (lcs)
En la segunda función, estamos comparando qué tan parecidas son dos palabras basándonos en sus letras:

Buscamos coincidencias: Queremos encontrar la "Máxima Subsecuencia Común" entre las cadenas "CADENA" y "COMUN".

No necesitan estar juntas: El algoritmo busca letras que aparezcan en ambas palabras en el mismo orden, aunque haya otras letras de por medio.

Resultado: Nos arroja que el tamaño es 2, porque ambas comparten las letras "C" y "N" en ese orden jerárquico.
